## 一、光的本质与波粒二象性

### 1.1 光的波动说与粒子说的历史争论

17世纪，牛顿（Newton）提出光的微粒说（光是由大量粒子组成的），惠更斯（Huygens）提出光的波动说（光是一种波）。这两种理论在此后两百年间互相竞争：

**波动说的证据**：光的干涉（双缝干涉）、衍射、偏振等现象，只有波才能解释。

**粒子说的困难**：波动说无法解释光电效应（经典波动理论预测光电子动能应随光的强度增大，与实验不符）。

**量子力学的统一**：爱因斯坦（Einstein）提出光量子（光子）假说，成功解释光电效应，并提出光具有波粒二象性（既是波又是粒子）。这一思想后来被德布罗意（de Broglie）推广到一切粒子（物质波）。

### 1.2 波粒二象性的核心内容

**光子的能量**：E = hf（h = 6.626×10⁻³⁴ J·s，普朗克常数；f 为光的频率）

**光子的动量**：p = h/λ（λ 为光的波长，E = pc 对光子成立）

**光的波长和频率**：c = λf（c = 3×10⁸ m/s，光速）

**波粒二象性的理解**：光频率越高（波长越短），每个光子能量越大（高能光子）；光的强度越大（单位时间内光子数越多），但每个光子能量不变。

**物质波**（德布罗意波）：一切运动的实物粒子都具有波动性，波长 λ = h/p = h/(mv)（p 为动量）。质量越大、速度越快，波长越短，波动性越不明显（宏观物体波长极短，波动性无法观测）。

## 二、光电效应

### 2.1 光电效应的实验现象

**光电效应**：金属表面受到光照射时，会有电子（光电子）从金属表面逸出的现象。

**实验规律（与经典波动说的矛盾）**：

**规律一（频率阈值）**：只有当入射光的频率超过某一最小值（极限频率 ν₀，也称截止频率）时，才能发生光电效应，与光的强度无关。

**规律二（瞬时性）**：光电效应几乎是瞬时发生的（时间小于10⁻⁹ s），与光的强度无关。经典波动说预测弱光须积累足够能量才能使电子逸出，需要一定时间，与实验矛盾。

**规律三（饱和电流与光强成正比）**：光电流的最大值（饱和光电流）与入射光的强度成正比（光越强，单位时间内光子数越多，逸出电子数越多）。

**规律四（最大初动能与频率有关）**：光电子的最大初动能与入射光的频率成正比，与光的强度无关。

### 2.2 爱因斯坦光电效应方程

爱因斯坦用光子理论完美解释了光电效应：光照射金属时，单个光子将全部能量 hf 转移给一个电子；电子须用 W（逸出功，克服金属对电子的束缚）克服金属对其的引力逸出，剩余能量转化为光电子的初动能。

**方程**：hf = W + Ek_max = hν₀ + (1/2)mv_max²

其中：h 为普朗克常数；f 为入射光频率；W = hν₀ 为逸出功（逸出功取决于金属本身）；ν₀ 为极限频率（最小照射频率，f < ν₀ 时不发生光电效应）；Ek_max = (1/2)mv_max² 为光电子的最大初动能。

**截止电压（遏止电压）**：在光电效应实验中，加反向电压 U 使光电流减为零时的最小电压，满足 eU = Ek_max（遏止电压与光的频率有关，与强度无关）。

### 2.3 光电效应的图像分析

**Ek_max - f 图像**：光电子最大初动能随频率增加而线性增大，斜率为 h（普朗克常数）；图线与 f 轴的交点为极限频率 ν₀；图线在 Ek 轴上的截距的绝对值为逸出功 W（图线与纵轴的交点在纵轴负半轴，截距 = -W）。

**I-U 图像（光电流-电压特性曲线）**：U > 0（正向电压）时，光电流达到饱和值（饱和光电流与光强成正比）；U < 0（反向电压）时，光电流逐渐减小，在遏止电压 -U₀ 时减为零；遏止电压 U₀ 与光强无关，只与频率有关（频率越高，U₀ 越大）。

## 三、氢原子能级与光谱

### 3.1 玻尔的氢原子模型

1913年，玻尔（Niels Bohr）提出氢原子的量子化模型，成功解释了氢原子光谱。

**玻尔模型的假设**：

**定态假设**：原子只能处于一系列不连续的能量状态（定态），电子在这些定态中做圆周运动而不辐射能量。

**跃迁假设**：原子从高能级（E_n，n 为量子数，较大）跃迁到低能级（E_m，m < n）时，释放一个光子；从低能级跃迁到高能级时，吸收一个光子；光子能量 hf = E_n - E_m（取正值）。

**氢原子能级公式**：E_n = -13.6/n² eV（n = 1, 2, 3, ...，n = 1 为基态，最低能级）

**各能级数值**：E₁ = -13.6 eV；E₂ = -3.4 eV；E₃ = -1.51 eV；E₄ = -0.85 eV；...；E_∞ = 0（电子被完全电离）。

### 3.2 氢原子能级跃迁与光谱

**发射光谱**：激发态氢原子向基态或低激发态跃迁时，释放光子，形成线状发射光谱。

**吸收光谱**：基态或低能级氢原子吸收特定频率的光子，跃迁到高能级，在连续光谱上留下暗线（吸收线）。

**高考常考计算**：

光子频率（或波长）：hf = E_n - E_m → f = (E_n - E_m)/h，λ = hc/(E_n - E_m)

能级跃迁的条件：入射光子能量 hf 须恰好等于某两能级之差，才能被吸收（不满足则不吸收）。

**氢原子激发态的光谱线数目**：处于第 n 能级的氢原子跃迁到所有低能级，可能产生的光谱线条数 = C(n,2) = n(n-1)/2。

### 3.3 能级与原子核外电子轨道

玻尔模型中，电子在圆形轨道上做圆周运动，量子数 n 越大，轨道半径越大（r_n = n²r₁，r₁ = 0.053nm 为玻尔第一轨道半径），电子离核越远，能量越高（但仍为负值，能量越接近零）。

**完全电离的条件**：若给氢原子基态（E₁ = -13.6 eV）提供至少 13.6 eV 的能量（电离能），电子可以完全脱离原子核（n→∞，E→0）。

## 四、核物理

### 4.1 原子核的组成与核力

**原子核**：由质子（p，带正电，质量约 1.007 u）和中子（n，不带电，质量约 1.008 u）组成。

**核力**：结合原子核中的核子（质子和中子）的短程强作用力，只在极短距离（约 10⁻¹⁵ m）内起作用，且核力比库仑排斥力强得多（约是库仑力的 100 倍），核力与电荷无关（质子与质子之间、质子与中子之间、中子与中子之间的核力相同）。

**质量数和原子序数**：质量数 A = 质子数 Z + 中子数 N；元素的化学性质由质子数（原子序数 Z）决定；同位素是质子数相同、中子数不同的原子（如 C-12 和 C-14）。

### 4.2 核衰变

**α 衰变**（发射 α 粒子，即 ⁴₂He）：

衰变规律：质量数减少4，质子数减少2。

⁴ᴬZ X → ⁽ᴬ⁻⁴⁾₍Z₋₂₎ Y + ⁴₂He

**β 衰变**（发射 β 粒子，即电子 e⁻）：

衰变规律：质量数不变，质子数增加1（中子转变为质子）。

⁴ᴬZ X → ⁴ᴬ₍Z₊₁₎ Y + ⁰₋₁e + 反中微子

**γ 射线**：伴随 α 或 β 衰变产生的高能光子辐射，不改变核的质量数和质子数，只携走多余的能量。

**半衰期 T₁/₂**：放射性元素中有半数原子核发生衰变所需的时间。公式：N(t) = N₀·(1/2)^(t/T₁/₂)（t 时刻剩余核数，N₀ 为初始核数）。

**注意**：半衰期是统计规律，描述大量原子核的平均行为，不能用于预测单个原子核的衰变时刻；半衰期不受温度、压力、化学状态等外界条件影响（只由核内部结构决定）。

### 4.3 核反应与核方程

**核反应的守恒律**：

质量数守恒（反应前后质量数之和不变）；

电荷数守恒（反应前后质子数之和不变）；

质能守恒（反应前后总能量守恒，质量亏损转化为能量）。

**高考重要核反应**：

卢瑟福发现质子：¹⁴₇N + ⁴₂He → ¹⁷₈O + ¹₁H

卢瑟福发现中子：⁹₄Be + ⁴₂He → ¹²₆C + ¹₀n

核裂变（铀-235）：²³⁵₉₂U + ¹₀n → ¹⁴¹₅₆Ba + ⁹²₃₆Kr + 3¹₀n + 大量能量

核聚变（氢弹/太阳能量来源）：²₁H + ³₁H → ⁴₂He + ¹₀n + 大量能量

### 4.4 质能方程与结合能

**爱因斯坦质能方程**：E = mc²（c = 3×10⁸ m/s）

**质量亏损**：原子核的质量略小于组成它的核子（质子+中子）质量之和，差值 Δm 称为质量亏损：Δm = Zm_p + Nm_n - M_核

**结合能**：将原子核拆散为单个核子所需的能量，等于质量亏损对应的能量：ΔE = Δm·c²

**比结合能**（平均每个核子的结合能）：比结合能越大，原子核越稳定；中等质量核（Fe附近）的比结合能最大，这是核聚变（轻核合并释能）和核裂变（重核分裂释能）都能释放能量的根本原因。

## 五、常见问题解答（FAQ）

**Q1：高考物理近代物理专题主要考察哪些内容？**

**A1：** 高考近代物理专题主要考察：光电效应（爱因斯坦方程、截止频率、遏止电压的计算，以及 Ek-f 图像的分析）；氢原子能级（能级图的识读，跃迁时光子能量的计算，电离能的求解）；核衰变（α 衰变和 β 衰变的规律，半衰期的计算，核反应方程的补全）；核反应（质量亏损和结合能的计算，核裂变和核聚变的对比）；以及波粒二象性的基本概念（光子能量公式 E = hf，物质波的波长 λ = h/p）。

**Q2：光电效应中，光的强度和频率各影响什么？**

**A2：** 光的频率和强度对光电效应的影响严格区分如下：频率的影响：决定能否发生光电效应（须 f ≥ ν₀，极限频率）；决定每个光电子的最大初动能（Ek_max = hf - W，频率越高，Ek_max 越大）；决定遏止电压（eU₀ = Ek_max，频率越高，U₀ 越大）。光强的影响：决定单位时间内照射到金属的光子数（强度大，光子数多）；决定饱和光电流（强度越大，单位时间内逸出的电子数越多，饱和电流越大）；对Ek_max 和遏止电压没有影响（强度不影响单个光子的能量）。

**Q3：玻尔模型中，氢原子能级公式是如何使用的？**

**A3：** 氢原子能级公式 E_n = -13.6/n² eV 的使用方法：代入量子数 n（1, 2, 3,...）求各能级的能量；计算跃迁时放出或吸收的光子能量：hf = |E_n - E_m|（高能级跃迁到低能级放出光子，低跃迁到高吸收光子）；求光子频率：f = |E_n - E_m|/h（注意单位换算，eV → J：1 eV = 1.6×10⁻¹⁹ J）；求电离能：从某能级电离到无穷远（E_∞ = 0）所需能量 = |E_n|（取绝对值）；判断是否能被电离：若入射光子能量 hf ≥ |E_n|，则处于第 n 能级的原子可以被电离。

**Q4：核衰变方程的补全，如何快速正确完成？**

**A4：** 补全核衰变/核反应方程的步骤：根据守恒律：反应两侧的质量数之和相等，电荷数之和相等；设未知粒子的质量数为 A，电荷数（质子数）为 Z；由守恒律列方程：A = 左侧质量数之和 - 右侧已知质量数之和；Z = 左侧电荷数之和 - 右侧已知电荷数之和；根据 A 和 Z 确定未知粒子：若 Z = 2, A = 4，是 α 粒子（⁴₂He）；若 Z = -1, A = 0，是 β 粒子（电子，⁰₋₁e）；若 Z = 0, A = 1，是中子（¹₀n）；若 Z = 1, A = 1，是质子（¹₁H）；若 Z = 0, A = 0，是 γ 光子（⁰₀γ）或中微子。

**Q5：半衰期如何计算？半衰期不受外界条件影响是什么意思？**

**A5：** 半衰期的计算公式：N(t) = N₀ × (1/2)^(t/T₁/₂)，其中 N₀ 为初始核数，N(t) 为 t 时刻剩余的未衰变核数，T₁/₂ 为半衰期。或者利用"经过 n 个半衰期后，剩余 N₀/2ⁿ"：若 t = nT₁/₂，则 N = N₀/2ⁿ，m = m₀/2ⁿ（质量同样按半衰期减少）。半衰期不受外界条件影响的含义：温度（从接近绝对零度到几千度）、压力（从大气压到极高压力）、化学形态（元素是否在化合物中）都不影响放射性同位素的半衰期。这是因为放射性衰变是原子核内部的自发核反应，与核外电子（决定化学性质）无关，外界物理化学条件无法影响核内部的结构。

**Q6：质量亏损与结合能的计算，有哪些常用换算？**

**A6：** 质量亏损与结合能的常用换算：原子质量单位：1 u = 1.66054×10⁻²⁷ kg；质能方程换算：1 u 对应的能量 = 1 u × c² = 1.66054×10⁻²⁷ × (3×10⁸)² = 1.494×10⁻¹⁰ J = 931.5 MeV（常用换算：1 u = 931.5 MeV/c²）；1 MeV = 10⁶ eV = 10⁶ × 1.6×10⁻¹⁹ J = 1.6×10⁻¹³ J。高考计算步骤：算出质量亏损 Δm（以 u 为单位）；ΔE = Δm × 931.5 MeV（直接用 1 u = 931.5 MeV 换算）；或 ΔE = Δm × c²（用 kg 单位时须先换算 Δm × 1.66054×10⁻²⁷）。

**Q7：核裂变和核聚变有什么区别？各自在什么条件下发生？**

**A7：** 核裂变（重核分裂）：重核（如 U-235）在中子轰击下分裂为两个中等质量核，同时释放 2 至 3 个中子和大量能量；可以引发链式反应（释放的中子继续引发更多裂变）；需要条件：中子轰击，并达到临界质量（维持链式反应的最小质量）；应用：核电站（可控链式反应）和原子弹（不可控链式反应）。核聚变（轻核合并）：轻核（如 H-2 和 H-3）在极高温高压下合并为较重核，释放大量能量；每个核子释放的能量比裂变更大；需要条件：极高温度（约 10⁷ K 以上，称为热核反应）和极高压力；应用：太阳和恒星的能量来源，氢弹（不可控），可控核聚变（托卡马克装置，尚在研究中）。

**Q8：γ 射线和 X 射线有什么区别？**

**A8：** γ 射线和 X 射线都是高频电磁波（高能光子），区别在于来源不同：γ 射线：来源于原子核内部的跃迁或衰变（核能级变化），频率通常高于 X 射线，能量更高，穿透力更强；X 射线（X-rays）：来源于高速电子撞击重金属靶，或原子内层电子的跃迁（电子壳层的能级变化，而非核级变化），频率略低于 γ 射线；应用上：γ 射线用于核医学、辐照杀菌、工业探伤；X 射线用于医学诊断（骨骼拍片）、机场安检、材料分析。

**Q9：波粒二象性中，频率高（波长短）的光有什么特点？**

**A9：** 频率高（波长短）的光的特点：每个光子能量大（E = hf，频率越高，能量越大）；波动性相对较弱（高频率光衍射和干涉效果不明显，因为波长短，衍射条件对狭缝尺寸要求极小）；粒子性相对明显（在光电效应、康普顿散射等粒子行为中表现更突出）；容易产生光电效应（因为单个光子能量大，更容易超过逸出功 W）。实例：紫外线（频率高于可见光）比可见光更容易引起光电效应；X 射线和 γ 射线频率极高，穿透性强，粒子性更突出。

**Q10：氢原子从高能级向低能级跃迁，产生的光子能量如何计算？如何判断是否在可见光范围？**

**A10：** 光子能量计算：hf = E_高 - E_低（取高能级减低能级的差值，取正值）；将各能级数值（E₁ = -13.6 eV，E₂ = -3.4 eV，E₃ = -1.51 eV，E₄ = -0.85 eV 等）代入；1 eV = 1.6×10⁻¹⁹ J，换算为焦耳后再求频率。判断是否在可见光范围：可见光频率约 4×10¹⁴ Hz（红光）至 7.5×10¹⁴ Hz（紫光），对应波长约 400nm（紫）至 700nm（红）；对应能量约 1.6 eV（红光）至 3.1 eV（紫光）；氢原子光谱中，落在 n=3,4,5,6 跃迁到 n=2 的系列（巴尔末系）中，部分线落在可见光范围（n=3→2：1.89 eV 红光；n=4→2：2.55 eV 绿光；n=5→2：2.86 eV 蓝光；n=6→2：3.02 eV 紫光）。

**Q11：β 衰变中，中子转化为质子，中子数减少1而质子数增加1，如何理解？**

**A11：** β 衰变的核内机制：β 衰变是原子核中一个中子自发转变为一个质子，同时放出一个电子（β 粒子）和一个反电子中微子（ν̄_e）。核反应：n → p + e⁻ + ν̄_e。宏观效果：质量数不变（中子数减少1，质子数增加1，A = Z + N 不变）；元素种类变化（原子序数 Z 增大1，变成新元素）；释放一个高速电子（β 射线）和反电子中微子（中微子很难探测，高考中通常不要求写出）。高考中书写 β 衰变方程时，通常只写出电子，中微子可以省略：ᴬ_Z X → ᴬ_(Z+1) Y + ⁰_(-1) e。

**Q12：光电效应遏止电压（截止电压）与光的频率的关系是什么？**

**A12：** 遏止电压 U₀ 与光的频率 f 的关系：由 eU₀ = Ek_max = hf - W = hf - hν₀，故 U₀ = (h/e)f - W/e = (h/e)f - (hν₀/e)；U₀ 与 f 成线性关系（U₀-f 图像为直线）；直线斜率 = h/e（通过实验测量斜率，乘以电子电荷 e，可以得到普朗克常数 h）；直线在 f 轴的截距 = ν₀（极限频率）；直线在 U₀ 轴的截距 = -hν₀/e = -W/e（逸出功对应的电压，负值）。应用：利用密立根实验（测量不同频率光的遏止电压）精确测定普朗克常数 h，是近代物理学的重要实验之一。

**Q13：原子从基态被激发到第 n 能级，最多能发射多少种频率的光？**

**A13：** 原子从第 n 能级向所有低能级跃迁（包括经过中间能级的多步跃迁），最多可能产生的不同频率光谱线数目 = C(n, 2) = n(n-1)/2。推导：在 n 个能级中，任意两个能级之间的跃迁都对应一种特定频率，共有 C(n,2) = n(n-1)/2 种组合。例：n = 4 时，最多 C(4,2) = 6 种频率（4→3，4→2，4→1，3→2，3→1，2→1）。注意：这是理论最大值，实际实验中不一定所有跃迁都会发生（取决于初始激发状态和粒子数分布）；对于"大量处于第 n 激发态的氢原子系统"，会产生全部 n(n-1)/2 种谱线。

**Q14：X 射线是怎么产生的？有什么特性？**

**A14：** X 射线的产生：高速电子（通过高压加速）轰击重金属靶（如钨）时产生：轫致辐射（X 射线连续谱）：高速电子在金属原子附近减速时，动能转化为 X 射线光子（能量连续分布）；特征 X 射线（标识谱线）：高速电子将金属内层电子打出，外层电子填补空位时，释放特定能量的 X 光子（谱线取决于靶材原子的能级）。X 射线的特性：频率范围约 10¹⁷ 至 10²⁰ Hz，波长约 0.01 至 10 nm；穿透力强（可穿透人体软组织，但被骨骼等高密度物质吸收）；能使底片感光（用于 X 光照片）；能电离气体，使荧光物质发光；不带电，不被电磁场偏转（与 α、β 粒子不同）。

**Q15：如何区分近代物理中各种射线（α、β、γ 射线和 X 射线）？**

**A15：** 各种射线的比较：α 射线：氦核（⁴₂He），带+2e 电荷，质量大，穿透力最弱（一张纸可挡住），电离能力最强，在磁场和电场中偏转（偏转较小，因质量大）。β 射线：高速电子（⁰₋₁e），带-e 电荷，质量小，穿透力中等（可穿过几毫米金属），电离能力中等，在磁场和电场中偏转显著（质量小，偏转大）。γ 射线：高频电磁波（光子），不带电，穿透力最强（需几十厘米铅板或数米混凝土才能屏蔽），电离能力最弱，在磁场和电场中不偏转。X 射线：与 γ 射线相似（均为电磁波，不带电，不偏转），但来源不同（γ 来自核跃迁，X 来自电子跃迁或轫致辐射），频率通常略低于 γ 射线。

**Q16：高考物理近代物理的选修模块有哪些？如何策略性选择？**

**A16：** 高考物理的选修模块（各省不同，以下以常见情况为例）包括：选修3-4（机械振动与波、光的折射衍射干涉、电磁波与相对论）；选修3-5（动量与冲量、核物理与近代物理）。策略性选择建议：选修3-5（核物理）：知识点相对独立（不依赖大量数学工具），公式记忆性强，得分较稳定，多数考生选择这一模块；选修3-4（振动与波）：内容涉及较多的数学（波的叠加、光的干涉条件），需要较强的空间想象力，若基础好可选；选择建议：若数学和几何直觉较强，可以尝试选修3-4；若希望通过记忆公式稳定得分，选修3-5（核物理+动量）通常更适合大多数考生。

**Q17：相对论的基本内容在高考中考哪些？**

**A17：** 爱因斯坦相对论在高考中的考察范围（选修3-4内容）：狭义相对论的两个基本假设（相对性原理：物理定律在所有惯性参考系中相同；光速不变原理：真空中光速在所有惯性参考系中均为 c）；相对论效应的定性描述：时间膨胀（运动时钟变慢：动钟变慢，"双胞胎悖论"）；长度收缩（运动方向上长度缩短：动尺变短）；质量增大（运动质量随速度增大）；质能方程：E = mc²（静止能量，不只是核反应的质量亏损，也适用于任何物体）；相对论在粒子物理中的应用（高能粒子加速器中的相对论效应）。高考通常只考相对论的定性理解，不进行复杂的相对论计算。

**Q18：放射性同位素在医学和工业中有哪些应用？**

**A18：** 放射性同位素的应用：医学应用：放射性标记（如 C-14 用于代谢研究，放射性碘用于甲状腺检查）；放射治疗（γ 射线或高能粒子束集中照射肿瘤，破坏肿瘤细胞）；PET 扫描（正电子发射断层扫描，利用正电子湮灭产生的 γ 光子对成像）；工业应用：辐照灭菌（用 γ 射线灭菌，适用于不耐热的医疗器械）；工业探伤（X 射线或 γ 射线检测焊缝内部缺陷）；厚度测量（利用 β 射线穿透率的变化实时监控金属板厚度）；年代测定：C-14 年代测定（利用大气中 C-14 的稳定含量和生物死亡后 C-14 的衰变，测定有机物的年代，误差约 ± 100 年，适用范围约 200 至 50000 年）。

**Q19：在近代物理中，电子伏特（eV）与焦耳（J）如何换算？计算中如何正确使用？**

**A19：** 电子伏特（eV）与焦耳（J）的换算：1 eV = 1.6 × 10⁻¹⁹ J（一个电子通过 1 伏特电位差加速时获得的动能）；1 keV = 10³ eV = 1.6 × 10⁻¹⁶ J；1 MeV = 10⁶ eV = 1.6 × 10⁻¹³ J。计算中的注意事项：能级差用 eV 表示时，须转换为 J 才能代入 hf（其中 h = 6.626 × 10⁻³⁴ J·s）；hf 的单位是 J，若知道 f 求 E，结果为 J，转换为 eV 需除以 1.6 × 10⁻¹⁹；质量亏损的能量用 MeV 比较方便（1 u = 931.5 MeV，直接乘以质量亏损的 u 数）。

**Q20：氢原子在不同能级时，电子的轨道速度如何变化？**

**A20：** 在玻尔模型中，随着量子数 n 增大（能级升高），电子轨道半径增大（r_n = n²r₁），电子速度减小（v_n = v₁/n，v₁ 为基态轨道速度约 2.19 × 10⁶ m/s）。直觉理解：轨道半径越大，电子离核越远，核对电子的引力越弱（按 1/r² 减小），向心加速度越小，因此速度越小（圆周运动 F = mv²/r，F 减小更快，v 减小）。动能变化：Ek = (1/2)mv²，动能正比于 v²，随 n 增大而减小（轨道越高，动能越小）；势能（负值，绑定能量）也随 n 增大而增大（绝对值减小，趋近于零）；总能量 E_n = -13.6/n² eV 随 n 增大而增大（趋近于零，越来越接近电离态）。

**Q21：在研究光电效应时，为什么用紫外线而不用红光做实验？**

**A21：** 紫外线比红光更容易引发光电效应的原因：紫外线频率高，单个光子能量大（E = hf，频率高则 E 大）；大多数金属的逸出功在 3 至 5 eV 范围（如铜的逸出功约 4.5 eV），对应的极限频率在紫外线范围内；紫外线光子能量（约 3.1 至 100 eV）超过逸出功，可以引发光电效应；可见红光光子能量约 1.6 至 2 eV，低于大多数金属的逸出功，无法引发光电效应；少数特殊材料（如铯，逸出功约 2.0 eV）可以在可见光（绿光或蓝光）下产生光电效应，但对红光通常也不足。

**Q22：链式反应是什么？维持链式反应的条件是什么？**

**A22：** 核链式反应：U-235 在中子轰击下裂变，释放 2 至 3 个中子，这些中子可以继续轰击其他 U-235 核引发裂变，以此形成连锁反应。维持链式反应的条件：临界质量（critical mass）：铀的质量须超过某一临界值，使每次裂变平均产生至少一个有效中子（引发下一次裂变），否则中子会从表面逸出而链式反应自行中止；中子速度：慢中子（热中子）更容易被 U-235 俘获，核电站使用减速剂（石墨或重水）将快中子减速为热中子；控制反应速率：核电站使用控制棒（含有 B-10 等强吸收中子的元素）调节中子数量，控制链式反应速率（防止失控）。原子弹：超临界质量的铀或钚，快速压缩或组合达到超临界状态，引发不可控链式反应（爆炸）。

**Q23：德布罗意物质波，在高考中如何考察？**

**A23：** 德布罗意物质波（物质波）在高考中的考察：基本公式：λ = h/p = h/(mv)（动量越大，波长越短）；定性分析：宏观物体（如球）的质量大，波长极短（如 0.1 kg 以 1 m/s 运动，λ = 6.6 × 10⁻³³ m，远小于可观测尺度），波动性无法观察；电子等微粒（质量小，速度快，但比宏观物体轻得多）的波长在纳米量级，可以通过电子衍射等实验观测；应用：电子显微镜（利用电子的波动性，波长比光短，分辨率更高，可以看到比光学显微镜更小的结构）；玻尔模型的量子化条件的解释：电子驻波（电子在轨道上形成驻波，轨道周长须是波长的整数倍，即 2πr = nλ = nh/p，正是玻尔轨道量子化条件的物质波解释）。

**Q24：核聚变为什么比核裂变释放更多的能量？**

**A24：** 核聚变每个核子释放能量更多的原因：能量与比结合能（每个核子的平均结合能）的变化量有关；轻核（如 H-2 和 H-3）的比结合能较低（约 1 至 3 MeV/核子）；中等质量核（如 He-4）的比结合能较高（约 6 至 8 MeV/核子）；核聚变：轻核合并为中等质量核，比结合能从约 2 MeV/核子增加到约 7 MeV/核子，每个核子释放约 5 MeV 能量；核裂变：重核（如 U-235，比结合能约 7.6 MeV/核子）分裂为中等质量核（比结合能约 8.5 MeV/核子），每个核子释放约 0.9 MeV 能量；因此核聚变每个核子释放的能量约是核裂变的 5 至 6 倍，但核聚变的条件（极高温度和压力）目前难以在地面可控实现。

**Q25：高考物理近代物理的备考建议是什么？**

**A25：** 近代物理备考的最终建议：熟记核心公式（E = hf，Ek_max = hf - W，E_n = -13.6/n² eV，r = mv/(qB) 等），做到在 30 秒内写出所有公式；掌握守恒律判断核反应（质量数守恒、电荷数守恒，用于补全核方程）；重点练习光电效应方程的计算应用（结合图像分析 Ek-f 图和 I-U 图）；系统学习氢原子能级（背好前几个能级的数值，能计算跃迁光子的能量和频率）；核物理部分重点掌握α和β衰变规律、半衰期计算、质能方程计算；利用[高考历年真题练习 - ReportMedic](https://reportmedic.org/tools/gaokao-previous-year-question-papers.html)系统刷近代物理历年真题，掌握高考命题的题型和难度，将备考成果最大化地转化为高考分数！高考物理近代物理，认真备考，必得高分！

## 六、近代物理综合练习与备考总结

### 6.1 近代物理核心公式速查

**光子能量**：E = hf = hc/λ（h = 6.626 × 10⁻³⁴ J·s）

**光电效应**：hf = W + Ek_max；W = hν₀；eU₀ = Ek_max

**物质波**：λ = h/p = h/(mv)

**氢原子能级**：E_n = -13.6/n² eV；跃迁：hf = |E_n - E_m|

**核衰变**：α 衰变：A减4，Z减2；β 衰变：A不变，Z加1

**半衰期**：N(t) = N₀ × (1/2)^(t/T₁/₂)

**质能方程**：E = mc²；ΔE = Δm·c²；1 u = 931.5 MeV
