## 一、化学反应速率

### 1.1 化学反应速率的定义与计算

**化学反应速率**：单位时间内某种物质浓度的变化量，用于描述反应快慢。

v(A) = Δc(A)/Δt（单位：mol/(L·s) 或 mol/(L·min)）

**不同物质的速率之比**：等于化学计量数之比。对反应 aA + bB → cC + dD：

v(A) : v(B) : v(C) : v(D) = a : b : c : d

**注意**：纯固体和纯液体（如水）的浓度为常数，不写入速率表达式和平衡常数表达式；溶液中的反应用浓度，气相反应用浓度或分压。

### 1.2 影响化学反应速率的因素

**浓度**：增大反应物浓度，速率加快（单位体积内粒子数增多，碰撞频率增大）。

**压强**（对气体反应）：增大压强，气体浓度增大（等效于增大浓度），速率加快。

**温度**：升高温度，速率加快（通常每升高10K，速率约增大2至4倍）；降低温度，速率减慢。

**催化剂**：催化剂降低反应活化能，显著加快反应速率（包括正反应和逆反应，不改变平衡位置）。

**固体颗粒大小**：固体颗粒越小，接触面积越大，反应速率越快（如粉末状比块状反应更快）。

## 二、化学平衡

### 2.1 化学平衡状态

**化学平衡**：在一定条件下，可逆反应的正反应速率和逆反应速率相等，反应物和生成物的浓度不再随时间变化的状态。

**平衡的特征**：动（正逆反应仍在进行，动态平衡）；等（正反应速率 = 逆反应速率）；定（各物质的浓度保持不变）；变（条件改变则平衡可能移动）。

**判断平衡状态的方法**：

若某条件使"各物质浓度不变"（混合气体颜色不变、气体密度不变、某反应物转化率不变等），则系统处于平衡状态；

若"v正 = v逆"且方向相同的比较（同一物质消耗速率 = 生成速率，即"一消一生"同一物质），则处于平衡；

若所有量均为"同类量"（均为消耗或均为生成），须换算后比较是否满足计量比。

### 2.2 平衡常数 K

对反应 aA(g) + bB(g) ⇌ cC(g) + dD(g)，平衡常数：

K = c(C)^c · c(D)^d / [c(A)^a · c(B)^b]（浓度平衡常数，各物质为平衡时的浓度）

**K 的特点**：只与温度有关（K 随温度变化，与浓度、压强无关）；纯固体和纯液体不出现在 K 表达式中；K 越大，正反应进行程度越大（平衡越偏向生成物）。

**多步反应 K 的关系**：若反应 C = 反应 A + 反应 B，则 K_C = K_A × K_B；若反应 C = 反应 A 的逆反应，则 K_C = 1/K_A；若反应 C = n × 反应 A，则 K_C = K_A^n。

**反应商 Q**：用于判断反应方向。Q 的表达式与 K 相同，但浓度为任意时刻（非平衡时）的值：Q < K，反应正向进行；Q = K，已达平衡；Q > K，反应逆向进行。

### 2.3 勒夏特列原理（平衡移动原理）

**内容**：如果改变影响平衡的某一条件（浓度、温度、压强），平衡将向着能够减弱这种改变的方向移动。

**浓度变化的影响**：增大反应物浓度或减小生成物浓度，平衡正向移动；减小反应物浓度或增大生成物浓度，平衡逆向移动。

**温度变化的影响**：升高温度，平衡向吸热方向移动；降低温度，平衡向放热方向移动（记忆：升温有利于吸热反应，降温有利于放热反应）。

**压强变化的影响**（对气体分子数不同的反应）：增大压强，平衡向气体分子数减少的方向移动；减小压强，平衡向气体分子数增多的方向移动；对气体分子数不变的反应，改变压强不影响平衡位置。

**催化剂的影响**：同等程度加快正逆反应速率，不影响平衡位置，但能缩短达到平衡的时间。

## 三、热化学

### 3.1 热化学方程式

**热化学方程式**：表示化学反应中能量变化的化学方程式，须注明各物质的状态（s=固态，l=液态，g=气态，aq=溶液中）和反应的焓变 ΔH。

**焓变 ΔH**：反应热（在恒压条件下的热量变化）；ΔH < 0，放热反应；ΔH > 0，吸热反应。

**书写要求**：

各物质状态须标注；

ΔH 的单位为 kJ/mol（注意：这里的"mol"指按方程式计量数进行反应时的热量，而非某一种物质的量）；

热化学方程式中的化学计量数可以是分数；

若反应逆向进行，ΔH 符号取反，绝对值不变。

### 3.2 盖斯定律

**内容**：化学反应的焓变只与反应的始态和终态有关，与反应的路径无关。

**应用**：通过已知反应的焓变，计算难以直接测量的反应的焓变（将已知反应组合加减，目标反应的 ΔH 等于对应组合的 ΔH 的代数和）。

**步骤**：写出目标反应；将已知反应通过倍乘和加减组合，使其中间步骤相消，只留下目标反应的反应物和产物；对应的 ΔH 进行相同的倍乘和加减操作。

**例**：已知：C + O₂ → CO₂，ΔH₁；2CO + O₂ → 2CO₂，ΔH₂。求：2C + O₂ → 2CO，ΔH。

ΔH = 2ΔH₁ - ΔH₂（将第一个反应乘2，减去第二个反应）。

## 四、电化学

### 4.1 原电池

**原电池**：将化学能转化为电能的装置。自发的氧化还原反应可以设计为原电池。

**组成要素**：两种活性不同的电极（通常是两种不同金属，或金属与非金属导体）；电解质溶液（能与电极材料反应或溶液能导电）；外电路连接（导线连接两极）。

**电极判断**：

负极：活性较强的金属，发生氧化反应（失去电子），是电子流出的一端；

正极：活性较弱的金属或非金属，发生还原反应（得到电子），是电子流入的一端；

电流方向（传统）：从正极经外电路到负极（与电子流向相反）；电解质中：阳离子向正极迁移，阴离子向负极迁移。

**铜锌原电池（丹尼尔电池）**：Zn（负极）|ZnSO₄溶液 || CuSO₄溶液 | Cu（正极）；负极反应：Zn - 2e⁻ → Zn²⁺（氧化）；正极反应：Cu²⁺ + 2e⁻ → Cu（还原）。

### 4.2 电解池

**电解池**：将电能转化为化学能的装置，使非自发的氧化还原反应发生。

**电极判断**：

阳极：与电源正极相连，发生氧化反应（阳离子逃离阳极？不，是阴离子在阳极被氧化）；

阴极：与电源负极相连，发生还原反应（阳离子在阴极被还原）。

**口诀**：阳极失去电子（氧化），阴极得到电子（还原）；阳极氧化，阴极还原（原电池正负极与电解池阳阴极的关系）。

**电解水**：2H₂O → 2H₂↑ + O₂↑（阳极产生 O₂，阴极产生 H₂，体积比 1:2）；电解质须选择不会在电极上放电的惰性电解质（如 H₂SO₄ 或 NaOH，H⁺和 OH⁻参与反应）。

**电镀**：以镀件为阴极，以镀层金属为阳极（阳极溶解补充金属离子），电解质溶液含镀层金属离子；阴极（镀件）上：金属离子被还原为金属，沉积在镀件表面；阳极（镀层金属）：金属被氧化溶解为离子，进入溶液（补充消耗的金属离子）。

### 4.3 法拉第电解定律

**内容**：在电解过程中，电极上析出（或溶解）的物质的物质的量与通过电极的电量（Q = It，电流乘以时间）成正比，与电子的转移数有关。

公式：m = (M/nF) × Q = (M/nF) × It（m 为析出质量，M 为摩尔质量，n 为电极反应中转移的电子数，F = 96500 C/mol 为法拉第常数）

**简化计算**：Q/F 得到电子的物质的量（mol），再根据电极反应计算析出物质的量。

## 五、水溶液中的离子平衡

### 5.1 水的电离与 pH

**水的自电离**：H₂O ⇌ H⁺ + OH⁻（Kw = c(H⁺) × c(OH⁻) = 10⁻¹⁴，25°C时）

**Kw 的性质**：Kw 只与温度有关（升温，水的电离程度增大，Kw 增大）；任何水溶液中 c(H⁺) × c(OH⁻) = Kw（恒成立）。

**pH 的计算**：pH = -lg c(H⁺)；酸性溶液（c(H⁺) > c(OH⁻)）：pH < 7；中性（c(H⁺) = c(OH⁻)）：pH = 7；碱性（c(H⁺) < c(OH⁻)）：pH > 7。

**强酸/强碱的 pH 计算**：强酸完全电离，c(H⁺) 直接等于酸的浓度；弱酸部分电离，须用电离常数 Ka 计算；混合溶液须用 c(H⁺) × c(OH⁻) = Kw。

### 5.2 弱酸弱碱的电离平衡

**弱酸电离**（以 HA 为例）：HA ⇌ H⁺ + A⁻；Ka = c(H⁺)·c(A⁻)/c(HA)（酸的电离常数）。

**弱碱电离**（以 MOH 为例）：MOH ⇌ M⁺ + OH⁻；Kb = c(M⁺)·c(OH⁻)/c(MOH)。

**电离程度的影响因素**：稀释（加水）促进电离（Ka 不变，但 c(HA) 降低，平衡正移，电离度增大）；升温促进电离（吸热过程，升温有利）；同离子效应（加入 H⁺ 或 A⁻，抑制 HA 的电离）。

### 5.3 盐类水解

**水解的本质**：盐中的弱酸根或弱碱阳离子与水反应，促进水的电离，使溶液显酸性或碱性。

**强酸弱碱盐**（如 NH₄Cl）：NH₄⁺ + H₂O ⇌ NH₃·H₂O + H⁺，溶液显酸性（pH < 7）。

**强碱弱酸盐**（如 CH₃COONa）：CH₃COO⁻ + H₂O ⇌ CH₃COOH + OH⁻，溶液显碱性（pH > 7）。

**强酸强碱盐**（如 NaCl）：不水解，溶液显中性（pH = 7）。

**弱酸弱碱盐**（如 CH₃COONH₄）：阴阳离子均水解，溶液酸碱性取决于两者水解程度的比较（CH₃COONH₄ 中，Ka ≈ Kb，溶液近乎中性）。

**促进水解的条件**：升温、加水稀释（均促进水解）。

**水解程度与电离程度的对比**：对弱酸的盐（如 CH₃COONa），水解和电离都发生，但水解程度远小于电离程度（溶液显碱性说明水解略占优）。

### 5.4 沉淀溶解平衡与 Ksp

**难溶电解质的溶解平衡**：对 AgCl(s) ⇌ Ag⁺(aq) + Cl⁻(aq)，溶度积：Ksp = c(Ag⁺) × c(Cl⁻)（纯固体不入 Ksp 表达式）。

**Ksp 的性质**：只与温度有关（不受浓度影响）；若离子积 Q_c < Ksp，沉淀溶解（或不沉淀）；Q_c = Ksp，处于溶解平衡；Q_c > Ksp，产生沉淀（或沉淀更多）。

**应用**：判断是否产生沉淀（将两种离子溶液混合，计算 Q_c，与 Ksp 比较）；沉淀完全的条件（一般认为离子浓度 < 10⁻⁵ mol/L 时，沉淀基本完全）；沉淀转化（用 Ksp 较小的沉淀替换 Ksp 较大的沉淀，即从较大 Ksp 向较小 Ksp 方向转化）。

## 六、常见问题解答（FAQ）

**Q1：高考化学反应原理大题的常见考查方向有哪些？**

**A1：** 高考化学反应原理大题的主要考查方向：化学平衡（平衡常数K的计算、平衡移动方向的判断、平衡转化率的计算、等效平衡的判断）；热化学（热化学方程式的书写、盖斯定律的应用、反应焓变与键能的计算）；电化学（原电池和电解池的正负极/阴阳极判断、电极反应方程式的书写、电解产物的判断、法拉第定律的计算）；离子平衡（pH计算、盐类水解方程式的书写、Ksp的应用、溶液中离子浓度的大小比较）。每年高考大题通常综合考察上述多个方向，须建立完整的知识体系。

**Q2：如何正确书写热化学方程式？常见错误有哪些？**

**A2：** 热化学方程式的书写规范：须标注各物质的状态（s、l、g、aq）；ΔH 须标注符号（放热为负，吸热为正）和单位（kJ/mol）；方程式中的化学计量数可以是分数；不能标注反应条件（如"加热"等），只标注ΔH。常见错误：漏写物质状态（最高频错误，一定不能省略）；ΔH 符号弄反（放热写正，吸热写负）；混淆热化学方程式中的系数（热化学方程式的系数改变时，ΔH 须等比例改变）；将"反应热"与ΔH混用（注意放热反应ΔH为负，但"反应热"通常说释放多少kJ，直接取正值描述）。

**Q3：平衡常数K和反应商Q有什么区别？如何利用Q判断反应方向？**

**A3：** K 是特定温度下反应达到平衡时浓度之比的常数；Q 是任意时刻（未必平衡）的浓度之比。判断反应方向：Q < K：反应物浓度偏大（或生成物浓度偏小），反应正向进行，直到 Q = K；Q = K：已达平衡，正逆反应速率相等；Q > K：生成物浓度偏大（或反应物浓度偏小），反应逆向进行，直到 Q = K。应用场景：判断一个非平衡状态的反应方向（代入当前浓度计算 Q，与 K 比较）；判断等效平衡（相同温度下，若两组条件下 Q 相同，则最终平衡状态相同）。

**Q4：勒夏特列原理中，哪些操作不会改变平衡常数K，哪些会？**

**A4：** 不改变K：改变浓度（增减反应物或生成物）；改变压强（对气体分子数相等的反应，压强不影响平衡位置，对不等的反应，压强影响平衡移动但K不变）；加入催化剂；改变固体或液体的用量。改变K：改变温度（K 是温度的函数，温度变化则K变化）。注意区分"平衡移动"和"K变化"：浓度或压强改变导致平衡移动，但K不变；只有温度改变才改变K。

**Q5：原电池的正极和负极如何判断？常见错误是什么？**

**A5：** 原电池正负极的判断方法：负极是活性较强（较活泼）的金属（在电化学序中位置较前），发生氧化反应（失去电子），质量通常减少；正极是活性较弱的金属或导体（不与电解质反应），发生还原反应（得到电子），质量通常增大（或不变）；电子从负极经外电路流向正极（电流方向相反）；溶液中阳离子向正极移动，阴离子向负极移动。常见错误：混淆原电池（正负极）和电解池（阴阳极）；认为正极发生氧化（实际是还原）；将电子流向与电流方向搞混；忘记判断外电路和内电路中的粒子流向。

**Q6：电解池阴阳极的放电顺序是什么？**

**A6：** 电解时，不同离子在电极上的放电（得失电子）有先后顺序，依照离子在电化学序中的位置：阳极（氧化，失电子）：阴离子的放电顺序（在含活性电极时，活性电极本身优先溶解氧化；含惰性电极时，阴离子放电顺序：S²⁻ > I⁻ > Br⁻ > Cl⁻ > OH⁻ > 含氧酸根（SO₄²⁻、NO₃⁻等最难）；阴极（还原，得电子）：阳离子的放电顺序（越易得电子的离子越先放电）：Ag⁺ > Hg²⁺ > Cu²⁺ > H⁺（酸性溶液中）> Pb²⁺ > Sn²⁺ > Ni²⁺ > Fe²⁺ > Zn²⁺ > H₂O中的H⁺（碱性/中性）> Al³⁺ > Mg²⁺ > Na⁺ > Ca²⁺ > K⁺；简记：越活泼的金属离子（越难被还原），排在越后面，越难在阴极析出。

**Q7：如何利用盖斯定律计算目标反应的ΔH？步骤是什么？**

**A7：** 盖斯定律计算的标准步骤：写出目标反应（清楚知道需要计算什么）；列出已知反应（每个已知反应都有对应的ΔH）；通过以下操作将已知反应组合为目标反应：将某个已知反应乘以系数n，对应ΔH乘以n；将某个已知反应逆向，ΔH符号取反；将若干个（经过上述操作的）反应相加，中间生成物和反应物互相抵消；目标反应的ΔH = 各已知反应ΔH的代数和（按相同的操作）。关键技巧：先看目标反应中每种物质在已知反应中出现的位置，确定各已知反应的正逆和倍数，然后代数相加。

**Q8：水溶液中，离子浓度的大小比较如何系统处理？**

**A8：** 水溶液中离子浓度大小比较的方法：首先判断溶液的酸碱性（强酸、强碱、盐的水解或电离）；确定溶液中的主要离子（及其来源）；利用电荷守恒（溶液中阳离子总正电荷 = 阴离子总负电荷）：对 NaA 溶液：c(Na⁺) + c(H⁺) = c(A⁻) + c(OH⁻)；利用质量守恒（某元素在溶液中各形态的总量守恒）；利用水电离关系：c(H⁺)·c(OH⁻) = Kw；综合以上关系，将各离子浓度排序。注意：弱酸弱碱盐溶液（如CH₃COONH₄）的比较须分析水解和电离的竞争，结论依赖于Ka和Kb的相对大小。

**Q9：pH计算中，稀释操作会如何影响强酸和弱酸的溶液？**

**A9：** 稀释对强酸和弱酸溶液pH的影响：强酸（如HCl）：稀释10倍，c(H⁺)减小10倍，pH增大1；稀释到极大（如加水无限稀释），pH趋近但不超过7（中性）；弱酸（如CH₃COOH）：稀释时，弱酸的电离度增大（更多弱酸分子电离），c(H⁺)减小但比同浓度强酸减小慢（因为新的H⁺从弱酸电离补充）；弱酸溶液稀释10倍，pH增大量小于1（因为电离度增大）；稀释到极大，pH也趋近于7。比较：相同 pH 的强酸和弱酸溶液，加水稀释同等倍数，弱酸溶液pH增大较少（c(H⁺)较大，更多H⁺可由电离补充）；弱酸的总浓度（分子+离子）大于强酸（pH相同时，弱酸浓度更大）。

**Q10：Ksp在沉淀转化中如何应用？**

**A10：** 沉淀转化的Ksp应用：沉淀转化的原理：用较难溶的沉淀（Ksp小）替换较易溶的沉淀（Ksp大）；向AgCl的悬浊液中加入NaI溶液，AgCl(白色)转化为AgI(黄色)，因为Ksp(AgI) < Ksp(AgCl)（AgI更难溶）；计算Q并与Ksp比较：Q_c = c(离子1)×c(离子2)；若Q > Ksp，生成沉淀；若Q < Ksp，沉淀溶解；沉淀完全的条件：通常要求离子浓度降至10⁻⁵ mol/L以下，此时认为该离子基本沉淀完全；沉淀的先后顺序：当溶液中有多种可被沉淀的离子时，先沉淀的是先达到Ksp的离子（加入沉淀剂时，Ksp最小的对应离子先沉淀）。

**Q11：如何书写电极反应方程式？**

**A11：** 书写电极反应方程式的步骤：确定电极的氧化还原类型（阳极/负极发生氧化，阴极/正极发生还原）；确定主要反应物（阳极：活性电极本身或溶液中的阴离子；阴极：溶液中的阳离子或H⁺/H₂O）；写出主要物质的变化（氧化态变化）；根据电荷守恒和原子守恒，补充 H⁺、OH⁻ 或 H₂O（酸性环境用H⁺，碱性环境用OH⁻，中性环境看具体情况）；在方程式左边标注失去或得到的电子数（写在箭头方向）；检验原子和电荷是否守恒。示例（电解CuSO₄溶液，惰性电极）：阳极（氧化）：2H₂O - 4e⁻ → O₂↑ + 4H⁺；阴极（还原）：Cu²⁺ + 2e⁻ → Cu。

**Q12：影响盐类水解程度的因素有哪些？**

**A12：** 影响盐类水解程度的因素：温度（升温促进水解，因为水解是吸热过程，，水电离是吸热过程，水解促进水的电离）；稀释程度（加水稀释促进水解，因为水解是可逆反应，稀释相当于减小生成物浓度，平衡正移）；溶液酸碱性（加酸抑制强碱弱酸盐的水解；加碱促进强碱弱酸盐的水解，类似向平衡中加入生成物）；对应酸（碱）的强弱（越弱的酸对应的酸根离子水解程度越大，如CO₃²⁻的水解程度大于CH₃COO⁻）。应用：加热蒸发Fe₂(SO₄)₃溶液会促进Fe³⁺水解，产生Fe(OH)₃和H₂SO₄，须在酸性条件下（加入硫酸）蒸发以防止水解。

**Q13：等效平衡的判断方法是什么？**

**A13：** 等效平衡指不同投料量但最终达到相同平衡状态（各组分浓度和转化率相同）。判断方法：对气体反应，将投料量等效转化为全是反应物（或全是生成物）；对反应前后气体分子数相等的反应（如H₂+I₂⇌2HI），等效转化后若投料量按化学计量比，则在相同温度和压强（或容积）下，最终达到等效平衡；对反应前后气体分子数不等的反应，须考虑压强的影响；恒容容器（固定体积）：等比例扩大投料量，等效于增大压强，再改变投料量到原来条件，得到等效平衡；恒压容器（固定压强）：等比例改变投料量，体积按比例变化，压强不变，若组分比例相同，则等效。等效平衡的实质：在相同条件下（T，P或T，V），若转化量相同，则平衡状态相同。

**Q14：活化能与催化剂的关系是什么？催化剂为何不改变平衡位置？**

**A14：** 活化能是反应物转化为产物所需克服的能量壁垒；催化剂通过提供反应的中间路径，降低正反应和逆反应的活化能，使更多分子具有足够能量发生有效碰撞，从而加快反应速率；催化剂同等程度降低正向和逆向活化能（正反应速率和逆反应速率均加快，且加快比例相同）；因此，催化剂不改变正反应速率与逆反应速率的比（即不改变平衡常数K），也不改变平衡位置（各物质的平衡浓度不变）；催化剂的作用：加快达到平衡的速度，而不改变平衡时各物质的浓度（只是"更快地"到达相同的平衡状态）。

**Q15：化学平衡中，转化率的计算需要注意什么？**

**A15：** 转化率计算的注意事项：转化率 = (某物质已反应的量)/(某物质初始的量) × 100%；同一反应中，不同反应物的转化率通常不同（取决于初始投料比与化学计量比的关系）；若投料比等于化学计量比，则所有反应物的转化率相同（对称投料）；若某反应物过量，则过量物质的转化率较低，不足量物质的转化率较高；增大某一反应物的用量，通常会提高其他反应物的转化率，而降低自身的转化率（使过量）；缩短反应器体积（增大压强）对气体分子数不等的反应有影响：压强增大，平衡向气体分子数减少的方向移动，使转化率变化（须具体分析）。

**Q16：如何判断溶液中离子方程式是否正确？**

**A16：** 判断离子方程式正确性的标准：原子守恒（方程式两边各元素的原子数相等）；电荷守恒（两边总电荷数代数和相等）；强酸、强碱、可溶性强电解质须拆写成离子形式；弱酸、弱碱、难溶物、水须保持分子形式不拆；气体须标↑，沉淀须标↓（在特定条件下）；量的关系正确（不能过量或不足量的写法，如Al与NaOH反应须按1:1计量）；常见错误：将弱酸（CH₃COOH、HF、H₂CO₃等）拆写成离子形式；将难溶盐（BaSO₄、AgCl等）拆写成离子形式；将水拆写成H⁺和OH⁻（在非电解情况下不应拆）；双水解反应漏写产物（如Al³⁺和CO₃²⁻发生双水解，须写出Al(OH)₃↓和CO₂↑）。

**Q17：化学平衡大题中，图像分析有哪些常见题型？**

**A17：** 化学平衡图像分析的常见题型：转化率-时间图：分析达到平衡的时间（影响速率的因素）；比较不同条件下平衡转化率的高低；判断改变条件对平衡的影响方向。浓度-时间图：判断各物质的消耗/生成量，确定化学计量比；确定平衡时各物质的浓度，计算K。速率-时间图：正逆速率曲线交叉点对应平衡时刻；改变条件时速率的突变和新平衡的建立。压强（或体积）对转化率的影响图：分析对气体分子数不等反应的影响；区分恒温恒压和恒温恒容条件下的不同结果。温度对K的影响：K随温度升高而增大，说明正反应为吸热；K随温度升高而减小，说明正反应为放热。

**Q18：弱酸弱碱中和后溶液的酸碱性如何判断？**

**A18：** 弱酸弱碱中和后溶液的酸碱性判断：若Ka > Kb（酸性比碱性强），弱酸的酸性更强，中和后酸根水解更弱、弱碱基阳离子水解更强，溶液显酸性；若Ka < Kb（碱性比酸性强），溶液显碱性；若Ka = Kb，溶液显中性（近似）；实例：CH₃COONH₄溶液，Ka(CH₃COOH) ≈ Kb(NH₃·H₂O) ≈ 1.8×10⁻⁵，两者相近，溶液显中性或极弱酸/碱性。判断依据：若弱酸的Ka > 弱碱的Kb，则酸更强，中和后形成的盐的弱碱基团水解更强（提供H⁺），显酸性；反之显碱性。

**Q19：高考化学反应原理大题，如何系统备考？**

**Q20：温度对化学反应的影响，在平衡和速率上有何不同？**

**A20：** 温度对速率和平衡的双重影响：对反应速率：升温总是加快正反应和逆反应速率（活化能决定，升温使更多分子达到活化能阈值）；对平衡位置：升温使平衡向吸热方向移动（只有这一方向的速率增大更显著，使正逆速率重新相等时，平衡向吸热方向偏移）；对平衡常数K：升温，若正反应为放热（ΔH < 0），则K减小（平衡逆移）；若正反应为吸热（ΔH > 0），则K增大（平衡正移）。实际应用：工业合成氨（N₂+3H₂⇌2NH₃，正反应放热）：升温加快速率（有利于尽快达到平衡），但降低NH₃的平衡转化率（不利于平衡产率）；工业上选择适中温度（500°C左右）是速率和转化率的折中。

**Q21：高考常见的反应原理计算题，有哪些高频题型？**

**A21：** 高考化学反应原理的高频计算题型：利用平衡常数K计算平衡浓度或转化率（设未知量x，列K表达式方程求解）；利用盖斯定律计算目标反应的ΔH（多步反应相加减）；利用法拉第定律计算电解量（Q=It，转换为电子摩尔数，再计算物质量）；利用pH和Kw计算溶液中H⁺或OH⁻浓度；利用Ksp计算溶液中离子浓度或判断沉淀是否产生；利用Ka或Kb计算弱酸弱碱溶液的pH（通常须近似处理：弱酸电离度较小时，分母近似为原浓度）。

**Q22：电解质溶液中，电荷守恒和质量守恒如何使用？**

**A22：** 在电解质溶液中：电荷守恒（任何电解质溶液中，所有阳离子的总正电荷 = 所有阴离子的总负电荷，每种离子须乘以其电荷数）：例：NaOH溶液：c(Na⁺) + c(H⁺) = c(OH⁻)（Na⁺是+1价，H⁺是+1价，OH⁻是-1价）；例：Na₂SO₄溶液：c(Na⁺) + c(H⁺) = 2c(SO₄²⁻) + c(OH⁻)（Na⁺是+1，H⁺是+1，SO₄²⁻是-2价须乘2）。质量守恒（某元素总量在溶液中守恒，无论以何种形式存在）：例：CH₃COOH水溶液中，碳酸的总量：c(CH₃COOH) + c(CH₃COO⁻) = 初始浓度（若是0.1 mol/L的CH₃COOH溶液，则 c(CH₃COOH) + c(CH₃COO⁻) = 0.1 mol/L）。

**Q23：在分析化学平衡图像时，如何判断什么时候达到平衡？**

**A23：** 判断化学平衡图像中的平衡时刻：各物质浓度不再随时间变化（浓度-时间图上，曲线变为水平线）；正反应速率与逆反应速率相等（v-t图上，正逆速率曲线重合）；特定可观测量不再变化（如：气体总压强不变、混合气体颜色不变、容积不变等，取决于具体情景）。注意："正反应速率等于逆反应速率"是平衡的充要条件，但须是"同种物质的消耗速率等于生成速率"，而非不同物质的速率比较；"浓度不变"是平衡的必要条件，但须在可逆反应的情境中（若反应进行到底，浓度也不变，但不是平衡）。

**Q24：如何区分可逆反应中的正向和逆向？**

**A24：** 区分可逆反应方向的依据：正反应：方程式书写的从左到右方向（消耗反应物，生成生成物）；逆反应：从右到左方向（消耗生成物，生成反应物）；正反应速率（v正）：单位时间内某反应物浓度减少量（或某生成物浓度增大量）；逆反应速率（v逆）：单位时间内某生成物浓度减少量（或某反应物浓度增大量）；平衡时 v正 = v逆：注意这是针对同种物质的"消耗速率等于生成速率"（不是两种不同物质的速率相等）；改变条件后的速率变化：所有使正反应速率加快的因素，同时也加快逆反应速率（催化剂、升温、增大浓度均如此），只是两者加快的比例可能不同，导致平衡移动。

**Q25：高考化学反应原理专题，最终的备考建议是什么？**

**A25：** 化学反应原理专题备考的最终建议：建立"四大模块"的完整知识框架（化学平衡、热化学、电化学、离子平衡），确保每个模块的核心规律掌握透彻；重点突破化学平衡（这是最难、分值最大的模块），通过大量练习内化K的计算、平衡移动判断、转化率计算的解题流程；电化学重点掌握电极反应方程式的书写（掌握"阳极失电子氧化、阴极得电子还原，酸碱环境补全"的书写框架）；离子平衡重点掌握pH计算和离子浓度排序（电荷守恒+质量守恒+水的电离三大关系）；每周做2至3道历年高考反应原理大题，积累对命题规律的感知，将理论知识转化为解题能力；利用[高考历年真题练习 - ReportMedic](https://reportmedic.org/tools/gaokao-previous-year-question-papers.html)系统刷历年反应原理相关真题，把每一分可以拿到的分数都稳稳收入囊中！高考化学反应原理，认真备考，必得高分！

## 七、化学反应原理综合备考总结

### 7.1 核心公式速查

**化学平衡**：K = c(C)^c·c(D)^d/[c(A)^a·c(B)^b]（平衡常数）；Q 与 K 比较判断方向；勒夏特列原理（改变条件，向减弱改变的方向移动）。

**热化学**：ΔH（负值放热，正值吸热）；盖斯定律（ΔH只与始末状态有关）。

**电化学**：原电池（负极氧化、正极还原）；电解池（阳极氧化、阴极还原）；Q = It（电量），n(e⁻) = Q/F。

**离子平衡**：Kw = c(H⁺)·c(OH⁻) = 10⁻¹⁴（25°C）；pH = -lg c(H⁺)；Ksp = c(阳离子)^m·c(阴离子)^n。

### 7.2 化学反应原理的学科价值
