## 一、立体几何在高考中的地位与考查结构

### 1.1 立体几何的分值分布

在高考数学试卷中，立体几何的分值分布：

**选择题/填空题中的立体几何：** 通常1-2道，涉及基本性质的判断（如命题真假、简单计算），每道4-5分，合计约8-10分；

**大题中的立体几何：** 一道大题，满分约12分，通常分为两小问：第一问：证明（线面平行、面面平行、线面垂直等）约4-6分；第二问：计算（二面角、线面角、体积等）约6-8分；

**备考优先级：** 立体几何大题的前一问（证明题），是相对容易且有规律的，应当成为稳定得分的目标；第二问的角度/体积计算，须掌握向量法或纯几何法；

### 1.2 立体几何的两大解法体系

高考立体几何有两种解法体系，各有优缺点：

**空间向量法（向量法）：** 通过建立空间直角坐标系，将几何问题转化为向量计算；优点：方法系统、不依赖几何直觉；缺点：建立坐标系有难度，计算量较大；

**纯几何法（传统法）：** 利用几何定理（垂直、平行等）直接推导；优点：对几何直觉好的学生更自然；缺点：证明路径依赖灵感，对复杂图形较难；

**选择建议：** 对于大多数学生，向量法更系统可靠（不需要"灵感"），建议优先掌握向量法；在向量法熟练的基础上，了解纯几何法的常用证明思路；

## 二、高考常见的三维图形

### 2.1 正方体与长方体（直平行六面体）

正方体和长方体是高考立体几何出现频率最高的图形：

**正方体的关键性质：** 所有棱长相等，所有面为正方形；每条棱与4条棱垂直（相邻面的棱），与4条棱平行（对面的棱），与4条棱既不垂直也不平行（体对角面上的棱）；体对角线长度 = √3 × 棱长（如棱长为1，体对角线长√3）；正方体的面对角线长度 = √2 × 棱长；

**常见考查方向：** 正方体上取若干点，证明某面与某棱平行/垂直；求两条棱或两个面之间的角（二面角）；求特殊三角形的面积（如面对角线构成的三角形）；

### 2.2 棱柱（三棱柱、四棱柱）

棱柱是高考另一类高频图形：

**三棱柱的特点：** 两个平行且全等的三角形底面，三个平行四边形侧面；直三棱柱的侧棱垂直于底面（高考最常见）；正三棱柱的底面是正三角形，侧面是矩形；

**四棱柱（含正四棱柱）的特点：** 底面为四边形，侧棱彼此平行；正四棱柱（正方形底面，侧棱垂直底面）是最常见的四棱柱；

### 2.3 棱锥（三棱锥、四棱锥）

棱锥在高考中也频繁出现：

**三棱锥（四面体）的特点：** 由四个三角形面围成；高考中常见的是"正三棱锥"（底面为正三角形，顶点在底面中心正上方）；

**四棱锥的特点：** 底面为四边形，四个三角形侧面；正四棱锥（底面为正方形，顶点在底面中心正上方）是最高频的四棱锥；

### 2.4 球（圆球）

球在高考中通常作为辅助几何体出现（如"正方体的外接球"、"正三棱柱的外接球"等）：

**关键性质：** 球的表面积 = 4πR²；球的体积 = (4/3)πR³；外接球（经过所有顶点）：需要找球心（外接球球心到各顶点距离相等）；内切球（与所有面相切）：需要找内切球心（等距离于所有面）；

**与其他几何体的关联：** 正方体的外接球直径 = 体对角线长 = √3 × 棱长；正三棱锥的外接球，须通过球心与顶点距离相等建立方程；

## 三、空间向量法：系统建立坐标系

### 3.1 建立空间直角坐标系的原则

向量法的关键第一步：在图形中找合适的位置建立直角坐标系。原则：

**优先以顶点为原点：** 通常选取有3条棱两两垂直的顶点作为原点（如正方体的任一顶点，正四棱锥底面中心上面建立的坐标系）；

**以三条互相垂直的棱（或方向）为坐标轴：** 正方体：三条相互垂直的棱为x、y、z轴；正三棱柱：底面的两个方向（选取便于计算的方向）和侧棱方向；

**直接读出关键点的坐标：** 建立坐标系后，所有顶点的坐标应当可以直接写出（不须计算，或只需简单运算）；

### 3.2 向量法处理不同几何问题

**判断线面垂直（向量法）：**
若向量a ⊥ 向量b 且 向量a ⊥ 向量c，且b、c不共线，则a ⊥ 由b、c确定的平面；计算：a·b = 0 且 a·c = 0，即法向量与面内两个方向向量的点积均为零；

**求两直线所成角（向量法）：**
设两直线的方向向量为a和b；cos θ = |a·b| / (|a||b|)（取绝对值，因为角度在0°到90°之间）；

**求线面角（向量法）：**
设直线方向向量为l，平面法向量为n；sin α = |l·n| / (|l||n|)（线面角用正弦）；

**求二面角（向量法）：**
求二面角的两个半平面的法向量n₁和n₂；二面角的余弦 cos θ = (n₁·n₂) / (|n₁||n₂|)；注意：须判断二面角是锐角还是钝角（法向量方向的选取影响结果的符号）；

### 3.3 向量法中的常见坐标系设置示例

**正方体（棱长为1）：** 以顶点A为原点，三条棱为坐标轴：A(0,0,0)，B(1,0,0)，C(1,1,0)，D(0,1,0)，A'(0,0,1)，B'(1,0,1)，C'(1,1,1)，D'(0,1,1)；

**正三棱柱（底面边长为2，高为h）：** 底面正三角形，以中心为原点：A(1,0,0)，B(-1/2,√3/2,0)，C(-1/2,-√3/2,0)（高考通常设更简单的坐标，如以底面某顶点为原点）；

**正四棱锥（底面边长为2，高为h）：** 底面中心O为原点，x轴和y轴沿底面两个方向：A(1,1,0)，B(-1,1,0)，C(-1,-1,0)，D(1,-1,0)，顶点P(0,0,h)；

## 四、纯几何法：推理技巧

### 4.1 线面平行的证明策略

**线面平行的判定定理：** 若直线l不在平面α内，且l与平面α内的某直线平行，则l∥α；

**证明思路：** 在平面α内找一条直线l'，证明l∥l'（利用平行四边形、中位线等性质）；

**常用辅助线策略：** 过某点作某直线的平行线；连接中点（利用三角形中位线定理）；过某点作某面的垂线；

### 4.2 线面垂直的证明策略

**线面垂直的判定定理：** 若直线l与平面α内的两条相交直线都垂直，则l⊥α；

**常用策略：** 若已知l⊥底面（正棱柱、正棱锥的情形），直接利用；利用三垂线定理：若直线在平面内的射影与平面内的某直线垂直，则原直线也与该直线垂直；

### 4.3 二面角的纯几何求法

**二面角的定义：** 两个半平面共享一条边（棱），所成的角；

**求法：** 在棱上取一点，分别在两个半平面内作棱的垂线，两垂线所成的角即为二面角；即"做二面角的平面角"：在棱上一点P，在两半平面内分别取PA⊥棱、PB⊥棱，则∠APB即为二面角；

**常见特殊二面角：** 正方体中，两相邻面的二面角为90°；正四棱锥的侧面与底面的二面角，须计算斜高与底面中点的关系；

## 五、体积计算专题

### 5.1 常见几何体的体积公式

**棱柱体积：** V = S底 × h（底面积乘以高）；

**棱锥体积：** V = (1/3) × S底 × h（底面积乘以高除以3）；

**球的体积：** V = (4/3)πR³；

**柱体与锥体的关系：** 等底等高的锥体体积 = 柱体体积的 1/3；

### 5.2 组合体的体积求法

部分高考题目给出的是几何体的一部分（如从棱锥中截去一个小棱锥），须用"大几何体体积减去截去部分"的方法：

**关键：确定截去部分的底面积和高；**

**相似体的体积比：** 若两个相似几何体的线性比例为k，则体积比为k³；

### 5.3 通过向量求四面体体积

四面体（三棱锥）的体积，可以通过向量叉积和混合积求得：

V = (1/6)|（向量AB × 向量AC）· 向量AD|（三个棱向量的混合积的绝对值除以6）；

这个公式在坐标给出后，可以直接计算，是向量法处理四面体体积的标准方法。

## 十、立体几何的高频题型专项解析

### 10.1 题型一：直线与平面的位置关系证明

**线面平行证明（高频）：**
在高考立体几何第一问中，"证明直线l∥平面α"是最高频的证明类型。标准步骤：在平面α内找一条直线l'，使l∥l'；利用线面平行判定定理：若直线l∥l'，且l'在平面α内，l不在平面α内，则l∥α；

关键技巧：寻找平行直线时，常用三角形中位线（连接两边中点，中位线平行于第三边）；平行四边形的对边；等比分割线（如DE∥BC，当D、E分别是AB、AC的等比分割点）；

**线面垂直证明（高频）：**
"证明直线l⊥平面α"的标准步骤：在平面α内找两条相交直线m和n；分别证明l⊥m和l⊥n；由线面垂直判定定理，l⊥α；

关键技巧：若图中有正棱柱或正棱锥，侧棱通常垂直于底面（直接利用）；对于不直观的情形，须寻找两个独立的垂直关系，再综合得出结论；

### 10.2 题型二：面面平行与面面垂直

**面面平行的证明：**
若平面α内有两条相交直线，分别平行于平面β，则α∥β；证明思路：在α内找两条相交直线l₁和l₂，分别证明l₁∥β和l₂∥β；

**面面垂直的判定：**
若平面α内有一条直线l⊥平面β，则α⊥β；证明思路：先证明某直线l⊥β，再说明l在α内，即得α⊥β；

### 10.3 题型三：线面角的求法

线面角（直线与平面所成的角），是直线在平面内的射影与直线本身的夹角：

**向量法：** 设直线方向向量为l，平面法向量为n；线面角α满足 sin α = |l·n| / (|l||n|)；

**几何法（三垂线定理）：** 作直线到平面的射影（垂足），连接射影与直线上某点，这条连线与射影的夹角即为线面角；

### 10.4 题型四：二面角的完整求法

二面角是高考立体几何大题第二问最常见的计算类型，须熟练掌握：

**步骤一：找到二面角的棱（两半平面的公共边）；**
**步骤二：在棱上取一点P；**
**步骤三：在第一个半平面内过P作棱的垂线PA；**
**步骤四：在第二个半平面内过P作棱的垂线PB；**
**步骤五：∠APB即为二面角（即"做二面角的平面角"）；**

向量法替代：求两个半平面的法向量n₁和n₂，cos θ = (n₁·n₂)/(|n₁||n₂|)；注意须判断结果是否须取补角。

## 十一、各类三维图形的高频考查模式

### 11.1 正方体的高频考法

正方体是立体几何出题频率最高的基础图形，以下是最常见的考查组合：

**组合一：正方体截面问题**
以正方体各面上的点为顶点，截取一个截面（如正三角形截面、正六边形截面等）；求截面面积、截面与底面的夹角；

**组合二：正方体内的线面关系**
取正方体的某些顶点或棱上的点，证明某直线∥某面或⊥某面；

**组合三：体对角线与面的夹角**
正方体体对角线（如A'C）与底面（ABCD）的夹角；解法：底面A'在底面的射影为A，∠CA'在底面的射影是AC，故线面角即∠CA'A，用tan∠CAA' = A'A/AC = 1/√2，即线面角=arctan(1/√2) = arctan(√2/2)；

### 11.2 正三棱柱的高频考法

**组合一：侧面上的点到底面的距离**
正三棱柱侧面上有点P，求P到底面（或某一侧面）的距离；

**组合二：截面问题**
过底面的某顶点和侧棱上的某点，作截面，求截面图形的形状和面积；

**组合三：外接球**
正三棱柱的外接球半径，须通过球心到各顶点距离相等的条件建立方程求解；

### 11.3 正四棱锥的高频考法

**组合一：侧面与底面的二面角**
最高频：正四棱锥底面边长为a，高为h，求侧面与底面的二面角；解法（纯几何）：侧面斜高m = √((a/2)² + h²)，二面角θ满足cos θ = (a/2)/m；

**组合二：两侧面之间的二面角**
相邻两侧面之间的二面角，棱为侧棱，需要找棱上的点和两侧面内的垂线；

**组合三：顶点到底面的距离（即高）的利用**
正四棱锥高h已知，可以计算体积V = (1/3)a²h，以及外接球半径等；

## 常见问题解答（FAQ）

**Q1：立体几何大题，应该用向量法还是纯几何法？**
对于大多数学生，建议优先使用向量法，原因：向量法有固定的步骤（建坐标系→写坐标→计算向量→点积或叉积），不依赖几何直觉；纯几何法虽然有时更简洁，但对复杂图形须要"灵感"找到合适的辅助线，容易卡壳；例外情形：若图形特别规整（如正方体），纯几何法可能更快；第一问（线面平行/垂直的证明），纯几何法通常更直观且步骤较少；第二问（角度和体积的计算），向量法更系统可靠。

**Q2：建立空间直角坐标系，如何选取原点和坐标轴方向？**
原则是让尽可能多的顶点坐标容易写出。通用策略：选取图形中有三条两两垂直的棱交汇的顶点作为原点；以三条垂直的棱的方向作为x、y、z轴；若没有直接垂直的棱，找辅助垂线（如正三棱柱的底面中心到顶点的方向可以作为一个坐标轴）。实践中，先画出图形，标出所有关键点，再决定最合适的坐标系设置。

**Q3：二面角的结果是钝角还是锐角，怎么判断？**
二面角的范围是0°到180°，所以可以是钝角。向量法中，若两个法向量的夹角θ满足cos θ > 0，则二面角为θ（若法向量方向选取合理）；若cos θ < 0，则二面角为180° - θ（即取补角）。判断依据：须结合图形的几何直觉，判断二面角应该是锐角还是钝角，然后选取对应的计算结果。若不确定，可以在计算后验证角度是否合理（如一个几何体内部的二面角通常<180°，外表面的二面角通常<180°）。

**Q4：正四棱锥的侧面与底面的二面角怎么求？**
设正四棱锥底面边长为2a，高为h，以底面中心为原点建立坐标系。斜高（底面中点到侧面顶点的距离）= √(a²+h²)（底面边长一半a加高h的勾股定理）；侧面与底面的二面角θ，满足tan θ = h/a（高与斜高底边之比）；若用纯几何法：在侧面内从顶点P到底面边AB的中点M作线段PM（侧面的斜高），再连接底面中心O到M（底面的斜高）；∠PMO即为侧面与底面的二面角（O为原点，M为底边中点）。

**Q5：三垂线定理是什么，什么时候使用？**
三垂线定理：若PA⊥平面α，AB在平面α内，BC也在平面α内且BC⊥AB，则PC⊥BC。即：斜线在平面内的射影AB若与平面内的直线BC垂直，则斜线AC也与BC垂直。使用场景：证明"斜线与平面内某直线垂直"时，先找斜线在平面内的射影，证明射影垂直于该直线，然后用三垂线定理得出斜线也垂直；求线面角时，通过三垂线定理找到线面角的角度（斜线与其在平面内射影的夹角）。

**Q6：如何求点到面的距离（在高考立体几何中）？**
有两种常用方法：方法一（向量法）：若已知平面方程（法向量n和平面内一点P₀），则点P到平面的距离 = |向量P₀P · n| / |n|；方法二（等体积法）：若已知整个几何体的体积和某个以目标面为底的锥体，则高（点到面的距离）= 3×体积/底面积；等体积法在高考中极为常用，因为体积通常比平面方程更容易求。

**Q7：立体几何大题的第一问（证明），有没有通用格式？**
有，以下是线面平行证明的通用格式示例：
"在△ABC中，取BC中点M，连接DM。由DM为△DBC的中位线，故DM∥BC且DM=BC/2。由题意，EF∥BC（或利用某条件建立EF∥BC），故DM∥EF。又因DM和EF不在平面α内，由线面平行判定定理，平面DM所在的面∥平面α，即DM所在直线∥平面α。"
关键格式要素：说明依据（哪个定理或条件）；说明结论（平行/垂直/共面等）；最后引用判定定理得出目标结论。

**Q8：正棱柱和正棱锥的区别是什么，高考中各自的考查重点是什么？**
正棱柱：底面是正多边形，侧棱垂直于底面，侧面是矩形；高考考查重点：侧面面积（侧棱×底面周长）、线面角（侧棱与底面对角线的夹角）、截面（用平面截正棱柱得到的截面形状）。正棱锥：底面是正多边形，顶点在底面中心的正上方，侧面是全等的等腰三角形；高考考查重点：斜高（侧面的高）的计算、侧面面积（斜高×底面边长×边数/2）、顶点到底面的距离（高）、二面角（侧面与底面、或两侧面之间）。

**Q9：立体几何中，"等积分割"是什么意思，怎么用？**
等积分割（等体积法）：将一个几何体看成两种不同的"底面+高"的组合，利用体积不变来建立方程求未知长度或角度。典型应用：已知四面体（三棱锥）的体积，求顶点到某个面的距离：V = (1/3) × S底 × h → h = 3V/S底；换底换高：同一个四面体，以不同面作为底，体积相等，可以建立方程求各面的面积关系；这种方法在高考题中极为常用，尤其是在已知整体体积、求某高度或某角度时。

**Q10：高考立体几何大题，建立坐标系后，写坐标时最常犯什么错误？**
以下是写坐标时最常见的错误：正三角形的顶点坐标写错：正三角形边长为a时，顶点坐标（以底边中点为原点）须通过勾股定理计算（高=√3a/2），不能随意写；混淆x、y、z轴的方向：建立坐标系后，须仔细核对"这个点应该在哪个轴的正方向"；正棱锥顶点坐标写错：正四棱锥底面中心为原点，顶点在(0,0,h)，而非在某个顶点上方；漏写某个顶点：写完所有顶点后，检查是否有遗漏。预防措施：建立坐标系后，先在草稿纸上对照图形逐个写出所有顶点的坐标并双重检查，再开始向量计算。

### 6.3 常见图形的直觉性质记忆

**正方体：** 体对角线互相平分；正方体内部有4条体对角线，每条长度=√3×棱长；面对角线长=√2×棱长；

## 十二、FAQ续集：立体几何深度追问

**Q11：向量法求二面角时，两个法向量的方向如何选取？**
法向量的方向有两种选择（正向和反向），不同选择会影响点积结果的正负，进而影响最终角度的判断。正确做法：求出点积cos θ后，结合图形的几何直觉判断二面角是锐角还是钝角；若点积结果为负（cos θ < 0），则两法向量夹角为钝角，但二面角可能是该钝角本身（当法向量方向选取合理时）或其补角；最安全的策略：计算|cos θ|，若图形直观上二面角应为锐角则θ = arccos|cos θ|，若应为钝角则θ = 180° - arccos|cos θ|。

**Q12：正三棱柱中如何快速找到底面正三角形的相关长度？**
正三角形边长为a的关键数据：高（中线）= √3a/2；从顶点到对边的距离（中线长）= √3a/2；内切圆半径 = a/(2√3) = √3a/6；外接圆半径 = a/√3 = √3a/3；从中心到顶点 = 外接圆半径 = √3a/3；从中心到边的中点 = 内切圆半径 = √3a/6。记住这些基本比例，在正三棱柱/正三棱锥建立坐标系时，可以直接写出坐标，不须每次重新推导。

**Q13：立体几何题中，已知球的内切或外接关系，如何求球的半径？**
外接球（经过所有顶点）：球心到每个顶点距离相等（都等于外接球半径R）；通过设球心坐标，建立到各顶点距离相等的方程组，求解球心坐标和R；内切球（与所有面相切）：球心到每个面的距离相等（都等于内切球半径r）；利用"体积 = 内切球半径 × 表面积 / 3"（类比圆的面积公式），可以不建立坐标系直接求r；具体：V = (1/3) × r × S表面积，故 r = 3V / S表面积。

**Q14：高考立体几何第一问总是"证明线面平行"，有没有速成的证明路径？**
有，以下是高频的证明路径：中位线路径：若直线DE是某三角形ABC两边的中点连线（D在AB，E在AC），则DE∥BC，DE可以用来建立线面平行；棱柱侧棱路径：棱柱的侧棱都平行，棱柱中的连线若与侧棱方向有关，可以快速得到平行；平行四边形路径：若题目给出某四边形为平行四边形，对边平行，可以直接利用；先找到目标直线的"平行替代物"（在目标平面内的平行直线），再用线面平行判定定理。

**Q15：体积计算中，"等积分割"的原理是什么？**
等积分割的核心：同一个几何体，用不同的"底面+高"的方式表达，体积不变，可以建立等式。最典型的应用：四面体ABCD，可以以面ABC为底，以D到面ABC的距离为高；也可以以面ABD为底，以C到面ABD的距离为高；两者体积相等，建立等式后求任意一个未知高度。还有一种用法：将大几何体分割为若干小几何体（已知体积），求某未知长度时，利用"体积之和等于总体积"建立方程。

## 十四、立体几何与物理的关联

### 14.1 立体几何与光学

光学中的反射定律（入射角等于反射角），在三维空间中涉及光线与反射面的夹角，这与立体几何中的线面角概念完全一致：

**平面镜反射：** 入射光线与反射面法线的夹角（入射角），等于反射光线与法线的夹角（反射角）；计算这些角度，就是立体几何中的线面角计算；

**实际高考考查：** 高考不直接考物理，但部分题目会以"光线照射到棱柱截面"为情境，实质上是立体几何的角度计算；

### 14.2 立体几何与工程设计

高考立体几何的部分"应用型"题目，会将几何体放入工程（如建筑屋顶、桥梁支架）或自然（如晶体结构）的情境中：

**读懂情境，提取几何模型：** 这类题目的第一步，是将文字描述的实际情境，转化为标准的棱柱、棱锥或球等几何模型；

**核心数学方法不变：** 无论情境如何，几何关系的求证和角度、面积、体积的计算方法，都与纯几何题相同；

## 十五、立体几何易错知识点完整清单

### 15.1 向量法的常见错误

**错误一：法向量计算错误**
求平面的法向量，须用面内两个不共线向量的叉积（或联立两个点积方程）；若平面由三点确定，先求两个面内向量，再用叉积或联立法求法向量；

**错误二：法向量方向选取后未验证**
二面角的结果须用图形直觉验证（是否符合几何上的直觉判断）；若数值结果看起来不合理（如结果是165°但图形明显是锐角），须检查法向量方向是否选取有误；

**错误三：线面角公式混淆**
线面角用正弦（sin α = |l·n|/(|l||n|)）；二面角用余弦（cos θ = n₁·n₂/(|n₁||n₂|)）；千万不能混淆这两个公式；

### 15.2 纯几何法的常见错误

**错误一：辅助线作图不准确**
"在棱上取点作二面角平面角"时，须确保两条辅助线都在各自的半平面内且都垂直于棱；若其中一条辅助线不垂直于棱，所作的角就不是二面角；

**错误二：三垂线定理的误用**
三垂线定理须满足：一条线（PA）垂直于平面，投影（AB）在平面内；另一条直线（BC）垂直于投影（AB）；结论才是：原斜线（AC）垂直于BC；不满足条件就直接套用定理是错误的；

**错误三：平行判定条件不充分**
线面平行的判定：须找到一条在平面内的平行直线，且这条直线必须确实在平面α内（而非平面β内）；若所找的平行直线不在目标平面内，推理是无效的；

## 十六、立体几何考前速查：核心公式与技巧

以下是高考前可以快速复习的立体几何核心知识点：

**体积公式：** 棱柱 V=Sh；棱锥 V=(1/3)Sh；球 V=(4/3)πR³；

**表面积公式：** 正棱柱 S=2S底+S侧（S侧=侧棱长×底面周长）；正棱锥 S=S底+S侧（S侧=(1/2)斜高×底面周长）；球 S=4πR²；

**证明常用定理：** 线面平行判定：直线∥面内一条直线→直线∥面；线面垂直判定：直线⊥面内两相交直线→直线⊥面；面面平行判定：面内两相交直线分别∥另一面→两面∥；面面垂直判定：面内一条直线⊥另一面→两面⊥；

**正方体关键数据（棱长a）：** 棱长a，面对角线√2a，体对角线√3a；

**正三角形关键数据（边长a）：** 高√3a/2，外接圆半径√3a/3，内切圆半径√3a/6；

## 二十一、FAQ第三批：立体几何实战进阶

**Q16：正四棱锥ABCD-P（P为顶点），如何快速建立坐标系写出所有顶点？**
标准设置：设底面正方形边长为2a，高为h。以底面中心O为原点，x轴沿AB方向，y轴沿AD方向，z轴沿OP方向建立坐标系；则各顶点坐标为：A(a,a,0)、B(-a,a,0)、C(-a,-a,0)、D(a,-a,0)、P(0,0,h)；若题目给出底面边长为a（不是2a），则A(a/2,a/2,0)等，须相应调整。用这套固定模板，可以在10秒内写出正四棱锥所有顶点坐标。

**Q17：立体几何中，已知两个面互相垂直，如何利用这个条件求角度？**
两面垂直（面面垂直）的条件，可以从以下角度利用：若α⊥β，且直线l在α内且l⊥α∩β（两面的交线），则l⊥β；这是一个很强的推论，可以从"面面垂直"直接得到"线面垂直"；在向量法中，若两面的法向量n₁和n₂满足n₁·n₂=0，则两面垂直；

**Q18：如何求两直线之间的距离（异面直线距离）？**
高考中异面直线距离的计算，通常用向量法：设异面直线l₁和l₂的方向向量分别为a和b，在l₁上取点P₁，在l₂上取点P₂；公共法向量n = a×b（叉积）；异面直线距离 = |向量P₁P₂ · n| / |n|；这个公式在高考中直接使用，不须推导。

**Q19：正三棱锥（底面正三角形ABC，顶点P在底面中心O正上方），PO⊥底面的证明怎么写？**
关键步骤：PA=PB=PC（正三棱锥定义，三条侧棱等长）；OA=OB=OC（O是正三角形外接圆圆心，即中心，等距于各顶点）；在△PAO中，PA²=PO²+OA²；在△PBO中，PB²=PO²+OB²；由PA=PB和OA=OB，两等式均成立；对于任意面内向量AX，△PAX中PA²=PO²+OA²，△PAX...（更严格的证明须展开）；高考中通常接受"由PA=PB=PC，O是底面中心，故PO⊥底面"的简略说明，但须说明OA=OB=OC（O是等距点）；

**Q20：立体几何大题建坐标系后，发现某顶点坐标很复杂（含√3等），计算容易出错，有什么解决办法？**
处理含√3坐标的策略：建议在计算向量时，先用字母代替复杂数值（如将√3/2记为m）进行代数化简，最后再代入具体数值；保留分数形式进行计算（避免小数带来的精度问题）；在各步骤之间，写出中间结果并检查符号；若图形对称，先利用对称性简化（如正三角形的某些计算可以利用对称性减少运算量）；最后，检验最终答案的量纲（角度在0°-180°之间，体积为正）；

**Q21：立体几何第一问证明和第二问计算，可以用不同的方法吗？**
完全可以，事实上这是很多学生的最优策略：第一问（线面平行/垂直证明）用纯几何法（定理推导，步骤简洁清晰）；第二问（角度/体积计算）用向量法（建坐标系，系统计算）；这样既保留了纯几何法在证明题上的简洁性，又利用了向量法在计算题上的系统性，是最高效的组合。

**Q22：向量叉积a×b在高中阶段是否可以直接使用？**
在高考标准课程中，向量叉积不是必修内容，高考评分标准不要求用叉积公式；但可以用联立方程的方式求法向量（设法向量(x,y,z)，通过面内两向量与法向量点积为零建立两个方程，联立求解），这是高考标准方法；若坚持使用叉积，计算结果正确即可，但须确保会用联立方程法（以防忘记叉积公式）；

**Q23：已知三棱锥四个顶点坐标，怎么快速求体积？**
最直接的方法是混合积公式：设顶点为A、B、C、D，V = (1/6)|（向量AB × 向量AC）· 向量AD|；其中叉积 AB × AC = (AB_y·AC_z - AB_z·AC_y, AB_z·AC_x - AB_x·AC_z, AB_x·AC_y - AB_y·AC_x)；再与AD点积（三个对应分量相乘后相加），取绝对值后除以6；若在高考中不使用叉积公式，可以用底面积×高/3的方法：先求某面的面积（用向量叉积的模，或手动计算三角形面积），再求顶点到该面的距离（用点到面距离公式）；

**Q24：立体几何的"斜截面"问题，有没有系统的处理方法？**
斜截面（用平面斜切几何体得到的截面）的处理方法：明确截面经过哪些顶点或棱上的哪些点；根据截面经过的点，确定截面的形状（三角形、四边形等）；用坐标法写出截面各顶点的坐标，计算截面面积；注意：斜截面的面积计算，通常须先用向量叉积（求截面两条边的叉积的模，即截面平行四边形面积），再根据截面形状做调整；

**Q25：高考立体几何大题，有没有"保底策略"（保证至少拿到一半的分）？**
以下是稳定得分的"保底策略"：第一问（证明题）：即便不会完整证明，也要写出"设…，由…定理知…"的框架（部分步骤有分）；建立坐标系（哪怕图形复杂，先画出来，写出尽可能多的顶点坐标）；完成向量计算的开头（写出关键向量，如AB、AC等）；即便最终结果错误，规范的中间步骤可以得到步骤分；高考立体几何大题通常有3-4个给分点（每个关键步骤各1-2分），不要因为"不会最后一步"就放弃前面的步骤分；

## 二十五、立体几何深度专题：向量法的高阶技巧

### 25.1 用联立方程求法向量（高考标准方法）

在不使用叉积公式的情况下，求平面法向量的标准方法：

设平面α由点A、B、C三点确定，则：面内两向量为向量AB和向量AC；设法向量n = (x, y, z)；列方程：n · 向量AB = 0 和 n · 向量AC = 0；这给出了关于x、y、z的两个方程（两个线性方程，三个未知数）；令z = 1（或其他非零值），解出x和y；即可得到一个具体的法向量；

**示例（正方体中求某面的法向量）：**
设正方体棱长为1，A(0,0,0)，B(1,0,0)，C(1,1,0)，D(0,1,0)，A'(0,0,1)；求平面A'BC的法向量：向量A'B = B - A' = (1,0,-1)；向量A'C = C - A' = (1,1,-1)；设n = (x,y,z)；n·(1,0,-1)=0：x-z=0，即x=z；n·(1,1,-1)=0：x+y-z=0；代入x=z：z+y-z=0，y=0；令z=1，则x=1，y=0；故法向量n=(1,0,1)；

### 25.2 二面角结果的正负判断

向量法求二面角时，最终结果的判断须注意：

cos θ = n₁·n₂ / (|n₁||n₂|)；若cos θ > 0，两法向量夹角为锐角，但二面角本身可能是锐角或钝角（取决于法向量的选取方向）；若cos θ < 0，两法向量夹角为钝角，二面角为该角或其补角；

**实用判断方法：** 结合图形直觉（凸多面体中大多数二面角是钝角，正方体面面之间是直角，正三棱锥侧面与底面之间通常是钝角）；或用两种不同的法向量方向计算，若结果互为补角，则根据直觉选择合理的那个；

**题目设置：** 正三棱柱ABC-A'B'C'，底面边长为2，高为√3，求侧棱AA'与底面对角线BC的夹角。

以A为原点，以AB方向为x轴，以底面内垂直AB方向为y轴，以AA'方向为z轴；底面正三角形ABC中，B(2,0,0)，C(1,√3,0)（由边长2和正三角形性质），A'(0,0,√3)，B'(2,0,√3)，C'(1,√3,√3)；

**第二步：写出相关向量**
方向向量：AA' = (0,0,√3)方向向量；BC的方向向量：C-B = (1-2,√3-0,0) = (-1,√3,0)；

**第三步：计算夹角**
cos θ = |AA'·BC_dir| / (|AA'||BC_dir|) = |(0,0,√3)·(-1,√3,0)| / (√3 × 2) = |0+0+0| / (2√3) = 0；
故夹角θ = 90°，即AA'与BC互相垂直（在正三棱柱中，侧棱确实垂直于底面所有方向的向量）；

### 31.2 正四棱锥建坐标系的完整示范

**题目设置：** 正四棱锥P-ABCD，底面边长为2，高PA=... 设高为h。求侧面PAB与底面ABCD的二面角。

**第一步：建坐标系**
以底面中心O为原点：A(1,-1,0)，B(1,1,0)，C(-1,1,0)，D(-1,-1,0)，P(0,0,h)；

**第二步：求底面ABCD的法向量**
底面ABCD法向量直接是z轴方向：n₂ = (0,0,1)；

**第三步：求侧面PAB的法向量**
面PAB内向量：向量AB = (0,2,0)，向量AP = (-1,1,h)；设n₁ = (x,y,z)，则n₁·(0,2,0)=0 → y=0；n₁·(-1,1,h)=0 → -x+0+hz=0 → x=hz；令z=1：x=h，y=0，n₁=(h,0,1)；

**第四步：求二面角**
cos θ = n₁·n₂ / (|n₁||n₂|) = (h,0,1)·(0,0,1) / (√(h²+1)×1) = 1/√(h²+1)；故二面角 θ = arccos(1/√(h²+1))；

**第一步：建坐标系（约2分钟）**

**第二步：写坐标（约2分钟）**

**第三步：建向量（约1分钟）**
根据题目要求，写出所需的方向向量或计算平面法向量；法向量用联立方程（令z=1，解x和y）；

**第四步：代入公式（约5分钟）**
线线角：cos θ = |a·b| / (|a|·|b|)；线面角：sin α = |l·n| / (|l|·|n|)；二面角：cos θ = n₁·n₂ / (|n₁|·|n₂|)；计算点积（对应分量乘积之和）；计算模长（各分量平方和开根号）；最终角度：arccos或arcsin计算结果；
